# экскурсия по городу

## 环城一游

### huán chéng yī yóu

||星期日有集市吗? / 市场是开着的吗？|xīngqírì yǒu jí shì ma? / shìchǎng shì kāizhe de ma?|рынок работает по воскресеньям?
||展览会星期一开放吗？|zhǎnlǎn huì xīngqí yī kāifàng ma?|ярмарка работает по понедельникам?
||展览会星期二开放吗？|zhǎnlǎn huì xīngqí'èr kāifàng ma?|выставка работает по вторникам?
||动物园星期三开放吗？|dòngwùyuán xīngqísān kāifàng ma?|зоопарк работает в среду?
||博物馆星期四开放吗？|bówùguǎn xīngqísì kāifàng ma?|музей по четвергам открыт?
||画廊星期五开放吗？|huàláng xīngqíwǔ kāifàng ma?|галерея открыта по пятницам?
||可以照相吗？|kěyǐ zhàoxiàng ma?|фотографировать можно?
||必须买门票吗？|bìxū mǎi ménpiào ma?|вход платный?
||门票多少钱？|ménpiào duōshǎo qián?|сколько стоит вход?
||对团体有优惠吗？|duì tuántǐ yǒu yōuhuì ma?|для групп есть скидка?
||对儿童有优惠吗？|duì er tóng yǒu yōuhuì ma?|для детей есть скидка?
||对大学生有优惠吗？|duì dàxuéshēng yǒu yōuhuì ma?|для студентов есть скидка?
||这座大楼是做什么用的？|zhè zuò dàlóu shì zuò shénme yòng de?|что это за здание?
||这座大楼建了多少年了？|zhè zuò dàlóu jiànle duō shào niánle?|сколько этому зданию лет?
||谁建的这座大楼？|shuí jiàn de zhè zuò dàlóu?|кто построил это здание?
||我对建筑很感兴趣。|wǒ duì jiànzhú hěn gǎn xìngqù.|я интересуюсь архитектурой.
||我对艺术很感兴趣。|wǒ duì yìshù hěn gǎn xìngqù.|я интересуюсь искусством.
||我对绘画很感兴趣。|wǒ duì huìhuà hěn gǎn xìngqù.|я интересуюсь живописью.
