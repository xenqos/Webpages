# прошедшая форма 3

## 过去时 3

### guò qù shí 3

||通电话（打电话）|tōng diànhuà (dǎ diànhuà)|говорить по телефону
||我打过电话了。|wǒ dǎguò diànhuàle.|я говорил / говорила по телефону.
||我一直在打电话了。|wǒ yīzhí zài dǎ diànhuàle.|я все время говорил / говорила по телефону.
||提问题，问问题，问|tí wèntí, wèn wèntí, wèn|спрашивать
||我问过了。|wǒ wènguòle.|я спросил / спросила.
||我以前经常问问题。|wǒ yǐqián jīngcháng wèn wèntí.|я всегда спрашивал / спрашивала.
||讲述、描述、告诉、说、叙述|jiǎngshù, miáoshù, gàosù, shuō, xùshù|рассказывать
||我说过了。|wǒ shuōguòle.|я рассказал / рассказала.
||我把整个事情都说过了。|wǒ bǎ zhěnggè shìqíng dōu shuōguòle.|я рассказал / рассказала всю историю.
||学习|xuéxí|учить
||我学习过了。|wǒ xuéxíguòle.|я учил / учила.
||我学习了整个晚上。|wǒ xuéxíle zhěnggè wǎnshàng.|я весь вечер учил / учила.
||工作|gōngzuò|работать
||我工作了。|wǒ gōngzuòle.|я работал / работала.
||我工作了一整天。|wǒ gōngzuòle yī zhěng tiān.|я весь день работал / работала.
||吃饭|chīfàn|есть
||我吃过了。|wǒ chīguòle.|я поел / поела.
||我把全部饭菜都吃光了。|wǒ bǎ quánbù fàncài dōu chī guāngle.|я съел / съела всю порцию.
