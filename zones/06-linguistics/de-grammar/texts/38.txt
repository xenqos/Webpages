# Приставки неотделяемые

|be-| ~~ ~~ |
>> %[m|besuchen] посещать


|ge-| ~~ ~~ |
>> %[m|gefallen] нравиться


|er-| ~~ ~~ |
>> %[m|erzählen] рассказывать


|ver-|перемещение~~удаление~~угасание|
>> %[m|versetzen] пересаживать
>> %[m|verstellen] переставлять
>> %[m|verschicken —] посылать


|zer-|разделение~~раздробление~~распад|
>> %[m|zerbrechen] разбивать
>> %[m|zerfallen] распадаться
>> %[m|zerschneiden] разрезать


|ent-|удаление~~отделение~~начало процесса|
>> %[m|entdecken] открывать
>> %[m|entspannen] расслабиться
>> %[m|entsprechen] соответствовать

[!
%[m|emp]- производная от %[m|ent]-
!]

|emp-| ~~ ~~ |
>> %[m|empfinden] ощущать
>> %[m|empfehlen] рекомендовать
>> %[m|empfangen] принимать


|miss-|недостаточность~~несовершенство~~неправильность|
>> %[m|misslingen] не удаться
>> %[m|missverstehen] неправильно] понимать
>> %[m|missbrauchen] неправильно] злоупотреблять


be-, er-: нет особых смысловых оттенков, но грамматически всегда дают аккузатив + прямое дополнение, транзитивны
ge-: обычно будет означать моментальное событие, которое отмечает начало или конец какого-то процесса — gefallen gestehen gedenken
