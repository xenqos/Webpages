Правила деления

  • количество слогов - количествo гласных
  • слог не может состоять из одного согласного
  • дифтонги ei, au, äu, eu, ai, oi, ui - один слог
  • длинные гласные ie, oo, aa, ee - один слог
  • слог может начинаться с гласной ver-ein-zelt, un-ver-ant-wort-lich
  • сочетания согласных обозначающих один звук - один согласный ch, sch, ck
  • удвоенные согласные и tz делятся Was-ser, Gi-raf-fe, Kat-ze
  • последний согласный в группе согласных - следующий слог Men-schen, man-chen, Ses-sel
  • составныe словa делятся по границам словa Glas-per-len, Pro-gramm-kode

Правила открытости

  • oткрытый слог оканчивается на гласную
  • закрытый слог оканчивается на согласную
  • условно-закрытый открываются при изменении формы слова
