# a la piscine

## в бассейне

|il fait chaud aujourd’hui.||сегодня жарко.
|si nous allions à la piscine ?||пойдём в бассейн?
|as-tu envie d’aller nager ?||у тебя есть настроение пойти поплавать?
|as-tu une serviette de bain ?||у тебя есть полотенце?
|as-tu une culotte de bain ?||у тебя есть плавки?
|as-tu un maillot de bain ?||у тебя есть купальник?
|sais-tu nager ?||ты умеешь плавать?
|sais-tu plonger ?||ты умеешь нырять?
|peux-tu sauter dans l’eau ?||ты умеешь прыгать в воду?
|où est la douche ?||где душ?
|où est la cabine pour se changer ?||где раздевалка?
|où sont les lunettes de plongée ?||где очки для плавания?
|est-ce que l’eau est profonde ?||здесь глубоко?
|est-ce que l’eau est propre ?||вода чистая?
|est-ce que l’eau est chaude ?||вода тёплая?
|je caille.||мне холодно.
|l’eau est trop froide.||вода слишком холодная.
|je sors maintenant de l’eau.||я сейчас выхожу из воды.
||
