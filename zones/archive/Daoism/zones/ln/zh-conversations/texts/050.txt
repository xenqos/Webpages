# в бассейне

## 在游泳馆里

### zài yóu yǒng guǎn lǐ

||今天天气很热。|jīntiān tiānqì hěn rè.|сегодня жарко.
||我们去游泳馆吗？|wǒmen qù yóuyǒng guǎn ma?|пойдём в бассейн?
||你有兴趣去游泳馆吗？|nǐ yǒu xìngqù qù yóuyǒng guǎn ma?|у тебя есть настроение пойти поплавать?
||你有毛巾吗？|nǐ yǒu máojīn ma?|у тебя есть полотенце?
||你有游泳裤吗？|nǐ yǒu yóuyǒng kù ma?|у тебя есть плавки?
||你有游泳衣吗？|nǐ yǒu yóuyǒng yī ma?|у тебя есть купальник?
||你会游泳吗？|nǐ huì yóuyǒng ma?|ты умеешь плавать?
||你会潜水吗？|nǐ huì qiánshuǐ ma?|ты умеешь нырять?
||你会跳水吗？|nǐ huì tiàoshuǐ ma?|ты умеешь прыгать в воду?
||淋浴在哪？|línyù zài nǎ?|где душ?
||更衣室在哪？|gēngyī shì zài nǎ?|где раздевалка?
||游泳眼镜在哪？|yóuyǒng yǎnjìng zài nǎ?|где очки для плавания?
||水深吗？|shuǐshēn ma?|здесь глубоко?
||水干净吗？|shuǐ gānjìng ma?|вода чистая?
||水暖和 / 温暖吗？|shuǐnuǎn huo/ wēnnuǎn ma?|вода тёплая?
||我感到很冷。|wǒ gǎndào hěn lěng.|мне холодно.
||水太凉了。|shuǐ tài liángle.|вода слишком холодная.
||我现在从水里出来。|wǒ xiànzài cóng shuǐ lǐ chūlái.|я сейчас выхожу из воды.
