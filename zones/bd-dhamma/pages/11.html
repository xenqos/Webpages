Game

Betting Rounds

Pre-Flop

'Small Blind' places small blind
'Big Blind' places big blind
'Button' deal 'Hole Cards' starting from left
If required all players place 'Ante'
Each player clockwise places: 'Fold' / 'Call' / 'Raise'
  • 'Big Blind' = 2 x 'Small Blind'
  • 'Call' = 'Big Blind'

Flop

'Button' deal 'Flop Community Cards'
'Small Blind' places: 'Fold' / 'Check' / 'Bet'
Each player clockwise places: 'Fold' / 'Check' / 'Bet' / 'Call' / 'Raise'

Turn

'Button' deal 'Turn Community Card'
'Small Blind' places: 'Fold' / 'Check' / 'Bet'
Each player clockwise places: 'Fold' / 'Check' / 'Bet' / 'Call' / 'Raise'

River

'Button' deal 'River Community Card'
'Small Blind' places: 'Fold' / 'Check' / 'Bet'
Each player clockwise places: 'Fold' / 'Check' / 'Bet' / 'Call' / 'Raise'

Showdown

Show 'Hole Cards'
The best 'Hand' wins

Raise & Re-raise

IF
  'Player' raises / re-raises
THEN
  All 'Players' starting from his left clockwise must match 'Raise' / 'Re-raise'
  Thus the betting stops at 'Player' to the right of the one who made 'Raise' / 'Re-raise'
IF
  'Players' >= 3 in 'Hand' ('Game')
THEN
  3-4 'Raise' / 'Re-raise' - allowed
ELSEIF
  'Players' = 2 are in 'Hand' ('Game')
THEN
  'Raise' / 'Re-raise' - not limited
